css y JavaScript

¿Qué es CSS?

CSS es un lenguaje utilizado para dar estilo a las páginas web, permitiendo la personalización de colores, fuentes, tamaños y disposición de los elementos.
CSS fue desarrollado por W3C (World Wide Web Consortium) en 1996. Se le conoce como Hojas de Estilo en Cascada porque las características se aplican de arriba a abajo.

Venatjas de CSS

  • Separa el contenido de la presentación, lo que facilita la gestión del aspecto visual de las páginas web
  • Permite a los diseñadores web realizar cambios en la apariencia de una web sin afectar al contenido
  • Ahorra tiempo al poder ordenar las preferencias de estilo en un único documento
  • Aumenta la velocidad de carga de las páginas

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite crear contenido dinámico e interactivo en los sitios web.

Caracteristicas principales

Interpreted

  • JavaScript se ejecuta directamente en el navegador web sin necesidad de compilación previa.

Orientado a objetos

  • Permite la creación de objetos y la gestión de sus propiedades y métodos.

Alto nivel

  • Es un lenguaje relativamente fácil de leer y escribir para los humanos.

Versatilidad

  • Se utiliza tanto para el desarrollo front-end (interfaz de usuario) como back-end (lógica del servidor) con frameworks como Node.js.

Asincronia

  • Permite realizar tareas en segundo plano sin bloquear la interfaz de usuario.

Ecosistema amplio

  • Cuenta con un gran número de librerías y frameworks que facilitan el desarrollo de aplicaciones complejas.

Multiplataforma

  • Se ejecuta en diferentes navegadores y sistemas operativos.

Tipado dinamico

  • El tipo de una variable se determina en tiempo de ejecución, no en tiempo de compilación.
